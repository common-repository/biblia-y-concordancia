This plugin allows the user to search different versions of the Bible and it includes audio.
Este plugin permite buscar en varias versiones de la Biblia e incluye audio.

== Bendicion.net ==
En Bendicion.net desarrollamos este plugin de WordPress con multiples versiones de la Biblia en espanol y en ingles con audio, y permite al usuario buscar en la Biblia por medio de libro, capitulo, versiculo y tambien por palabras en la concordancia. Los capitulos tambien se pueden ver en forma paralela. 
Visita Bendicion.net

== View Demo / Ver demo ==  
* **Espanol [https://bendicion.net/wp-biblia/](https://bendicion.net/wp-biblia/)**
* **English [https://bibleplugin.org/](https://bibleplugin.org/)**

* **Sube el plugin 'biblia y concordancia' al directorio `/wp-content/plugins/`.**
* **Activa el plugin por medio del menu 'Plugins' en WordPress.**
* **Agrega el codigo `[bendicion_biblia]` en una pagina nueva o tambien en un post.**
* **Tambien puede ser activado al agregar este codigo en un template:
   `<?php if (function_exists('display_bible_form')) { display_bible_form(); } ?>`**
* **El plugin también puede ser usado por medio de los widgets en `Apariencia > Widgets > Biblia y Concordancia`**
* **Si tiene una pregunta tecnica de la instalacion puede contactarnos a: info@bendicion.net**

== Características ==

* Búsqueda por palabras en concordancia
* Audio Biblia
* Incluye múltiples versiones bíblicas 
* Palabras de Cristo en letra roja
* Vista en paralelo con 2 versiones simultaneas
* Vista en paralelo con 3 versiones simultaneas
* Opción para cambiar el tamaño de la letra
* El plugin es compatible con WordPress Multisite
* Compatible con sitios web que usan SSL o https
* Incluye Comentario por Capítulo

== Búsqueda por Palabras / Search by Words ==

* Buscar Por Libro Individual / Search by individual book
* Buscar en el Antiguo Testamento / Search in the Old Testament
* Buscar en el Nuevo Testamento / Search in the New Testament

== Installation ==
* Sube el plugin 'biblia y concordancia' al directorio `/wp-content/plugins/`.
* Activa el plugin por medio del menu 'Plugins' en WordPress.
* Agrega el codigo `[bendicion_biblia]` en una pagina nueva o tambien en un post.
* Tambien puede ser activado al agregar este codigo en un template:
   `<?php if (function_exists('display_bible_form')) { display_bible_form(); } ?>`
* El plugin también puede ser usado por medio de los widgets en `Apariencia > Widgets > Biblia y Concordancia`
* Si tiene una pregunta tecnica de la instalacion puede contactarnos a: info@bendicion.net

Si no usas WordPress también puedes instalar la Biblia en tu página web copiando y pegando el siguiente código de HTML:

`<div id="api-selector"></div>`
`<script src="https://bendicion.net/biblia/jquery.min.js"></script>`
`<script src="https://bendicion.net/biblia/api-form.js"></script>`
